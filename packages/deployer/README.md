# @nexusforge/deployer

NexusForge Smart Deployer — Multi-cloud deployment, IaC generation, CI/CD pipeline builder.

## Installation

```bash
npm install -g @nexusforge/deployer
```

## Usage

```bash
nxf-deploy --help
```

## Features

- Multi-cloud deployment (Vercel, Netlify, AWS, GCP, Docker)
- Infrastructure as Code generation
- CI/CD pipeline builder
- Health checks and rollback
- Cloud provider detection
