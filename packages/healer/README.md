# @nexusforge/healer

NexusForge Self-Healing Engine — Autonomous bug detection, root cause analysis, and auto-fix PRs.

## Installation

```bash
npm install -g @nexusforge/healer
```

## Usage

```bash
nxf-heal --help
```

## Features

- Autonomous bug detection
- Root cause analysis
- Automated fix generation
- Continuous monitoring
