# @nexusforge/testgen

NexusForge AI Test Generator — Smart test generation, coverage analysis, and mutation testing.

## Installation

```bash
npm install -g @nexusforge/testgen
```

## Usage

```bash
nxf-test --help
```

## Features

- AI-powered test generation
- Coverage analysis
- Mutation testing
- Multi-framework support (Vitest, Jest, pytest)
