# @nexusforge/sdk

NexusForge Plugin SDK — Build extensions, integrations, and custom tools for the NexusForge ecosystem.

## Installation

```bash
npm install @nexusforge/sdk
```

## Usage

```javascript
import { Plugin, hooks, events } from '@nexusforge/sdk';
```

## Features

- Plugin architecture
- Event system
- Hook registration
- Plugin loader and registry
