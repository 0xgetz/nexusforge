# @nexusforge/cli

NexusForge AI Coding Assistant — Interactive CLI with multi-model support.

## Installation

```bash
npm install -g @nexusforge/cli
```

## Usage

```bash
nexusforge --help
nxf --help
```

## Features

- Interactive AI coding assistant
- Multi-model support
- Context-aware code generation
- Project scaffolding
