# @nexusforge/scanner

NexusForge AI Security Scanner — Dependency auditing, CVE lookup, and automated reports.

## Installation

```bash
npm install -g @nexusforge/scanner
```

## Usage

```bash
nxf-scan --help
```

## Features

- Dependency vulnerability scanning
- CVE database lookup
- Automated security reports
- Severity-based filtering
