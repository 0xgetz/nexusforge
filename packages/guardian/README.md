# @nexusforge/guardian

NexusForge Code Guardian — AI code review, quality metrics, architecture analysis, docs generation.

## Installation

```bash
npm install -g @nexusforge/guardian
```

## Usage

```bash
nxf-guard --help
```

## Features

- AI-powered code review
- Quality metrics analysis
- Architecture analysis
- Documentation generation
- Changelog generation
- Code diff analysis
- Custom rules engine
